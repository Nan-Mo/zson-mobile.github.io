## Install

use `npm`

```node
npm install @zson-design/cli --save-dev
```

or use `yarn`

```node
yarn add @zson-design/cli --dev
```

## Usage

```node
zson build [options]       打包编译仓库
zson dev [options]         运行开发环境
zson deploy [options]      部署官网站点
zson add [ComponentName]   新增组件模板
zson --help                查看帮助信息
zson --version             查看版本信息
```
