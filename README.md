## 浏览器支持

- iOS
- Android 4.0+

## 安装

npm 或 yarn 安装

```bash
# npm
$ npm install zson-mobile --save

# yarn
$ yarn add zson-mobile
```

## 使用

```js
import { Button, Cell } from 'zson-mobile';
import 'zson-mobile/dist/zson-mobile.css';
```
