<h1 align="center">Zson Design Mobile</h1>

<div align="center">

基于 React、React-Native 研发的一款移动端 UI 组件库。

</div>


## 浏览器支持

- iOS
- Android 4.0+

## 安装

npm 或 yarn 安装

```bash
# npm
$ npm install zson-mobile --save

# yarn
$ yarn add zson-mobile
```

## 使用

```js
import { Button, Cell } from 'zson-mobile';
import 'zson-mobile/dist/zson-mobile.css';
```
